Website anatomy

HTML (Hypertext Markup Language):

HTML bildet die Struktur der Webseite. Es verwendet Tags, um den Inhalt der Seite zu kennzeichnen und zu organisieren. In unserem Beispiel haben wir die grundlegende HTML-Struktur verwendet, einschließlich des <!DOCTYPE html>-Deklarationstags, des html-Tags als Wurzelelement und des body-Tags für den sichtbaren Inhalt der Seite.

CSS (Cascading Style Sheets):

CSS ermöglicht die Gestaltung und das Layout der Webseite. Es definiert Farben, Schriftarten, Abstände und andere visuelle Eigenschaften. In unserem Beispiel haben wir den Hintergrund auf schwarz (background-color: #000) gesetzt und die Textfarbe auf weiß (color: #fff) für eine kontrastreiche Darstellung. Die Übergangseffekte (transition) wurden verwendet, um sanfte Animationen beim Farbwechsel zu erzeugen.

JavaScript:

JavaScript macht Webseiten interaktiv, indem es auf Benutzeraktionen wie Klicks und Tastatureingaben reagiert. Es ermöglicht die Manipulation von Webseitenelementen und die Verarbeitung von Daten. Zudem können asynchrone Anfragen an den Server gesendet werden, um Daten zu laden oder zu speichern. Mit JavaScript können Sie Ihre Webseite lebendiger gestalten und eine interaktive Benutzererfahrung bieten.
